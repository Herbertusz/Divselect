A readonly és disabled attribútumok hatásait figyelhetjük meg ezeken a példákon.

Előre megadott attribútumok

				<select id="example1" name="example1">
					<option value="1">HTML</option>
					<option value="2">CSS</option>
					<option value="3" readonly="readonly">JavaScript</option>
					<option value="4">PHP</option>
					<option value="5">SQL</option>
					<option value="6" disabled="disabled">Python</option>
					<option value="7" disabled="disabled">Ruby</option>

				</select>

Readonly és disabled tulajdonságok változtatása

Az első sorban lévő gombokkal a select tulajdonságát változtathatjuk, a második sorban a harmadik (javascript) option-ét.

select:
option:

Ebben az esetben a HTML és javascript kód megegyezik (kivéve hogy sehol sincs readonly és disabled attribútum). A gombok HTML kódja, és az eseménykezelők: 

				<input id="selectdisable" type="button" value="disabled ki-bekapcsolása" />
				<input id="selectreadonly" type="button" value="readonly ki-bekapcsolása" />
				<input id="optiondisable" type="button" value="disabled ki-bekapcsolása" />
				<input id="optionreadonly" type="button" value="readonly ki-bekapcsolása" />
				$("#selectdisable").click(function(){
					if ($("#example2").divselect("isDisabled")){
						$("#example2").divselect("unsetDisabled");
					}
					else{
						$("#example2").divselect("setDisabled");
					}
				});
				$("#selectreadonly").click(function(){
					if ($("#example2").divselect("isReadonly")){
						$("#example2").divselect("unsetReadonly");
					}
					else{
						$("#example2").divselect("setReadonly");
					}
				});
				$("#optiondisable").click(function(){
					if ($("#example2").divselect("isDisabled", "3")){
						$("#example2").divselect("unsetdisabled", "3");
					}
					else{
						$("#example2").divselect("setDisabled", "3");
					}
				});
				$("#optionreadonly").click(function(){
					if ($("#example2").divselect("isReadonly", "3")){
						$("#example2").divselect("unsetReadonly", "3");
					}
					else{
						$("#example2").divselect("setReadonly", "3");
					}
				});
