Divselect

Ez a jQuery plugin elsősorban a HTML <select> elemének lecserélésére használható. A plugin igen rugalmas és testreszabható, így más célokra is alkalmas, például egy lenyíló menü is megvalósítható vele.
Egy select elemre lefuttatva a plugin lecseréli az elemet (alapértelmezett esetben) egy div-ekből álló, a select elemmel megegyező módon működő szerkezetre. A szerkezetbe helyez egy hidden inputot (rejtett űrlapelemet), ami a select által kiválasztott adat továbbítására szolgál. A pluginhoz tartozik egy CSS fájl is, amivel a kinézet tetszőlegesen testreszabható.
A plugin rendelkezik a select minden kényelmi funkcionalitásával, mint például a kurzorbillentyűkkel történő lépegetés, gyorskeresés karakterek leütésével, stb. Bővebben lásd a billentyűvezérlés részben.

  • Jelenlegi verzió: 1.0.0
  • Függőségek: jQuery, jQuery UI position
  • Szükséges: jQuery 1.7+, jQuery UI position 1.9+
  • Böngészők: IE7+, FF4+, Chrome, Opera11+, (IE6 alatt billentyűfunkciók kikapcsolása esetén)
  • Letöltés:
    • teljes csomag (zip)
    • jquery.divselect-1.0.0.js (fejlesztői változat)
    • jquery.divselect-1.0.0.min.js (tömörített változat)
    • divselect.css (alap CSS fájl)

Demó

Itt csak a legalapvetőbb funkciókat láthatjuk. Egy select elemre ráengedve konfigurálás nélkül: 

	$("#select-1").divselect();
HTML kód
		<select id="select-1" name="languages">
			<option value="html">HTML</option>
			<option value="css">CSS</option>
			<option value="js">JavaScript</option>
			<option value="php">PHP</option>
			<option value="sql">SQL</option>
			<option value="asp">ASP</option>
			<option value="vb">VBScript</option>
		</select>

Egy select optgroup tegekkel: 

	$("#select-2").divselect();
HTML kód
		<select id="select-2" name="languages">
			<option value="c">C</option>
			<option value="cplusplus">C++</option>
			<optgroup label="WEB - kliens oldal">
				<option value="html">HTML</option>
				<option value="css">CSS</option>
				<option value="js" selected="selected">JavaScript</option>
				<option value="vb">VBScript</option>
			</optgroup>
			<optgroup label="WEB - szerver oldal">
				<option value="php">PHP</option>
				<option value="sql">SQL</option>
				<option value="asp">ASP</option>
			</optgroup>
		</select>

A divselect plugin legjelentősebb funkciója, ami a select elem lehetőségeit felülmúlja az, hogy tetszőleges HTML kód beszúrható az option tegekbe. Az egyik paraméterrel felülírhatjuk az egyes option tegek belsejét tetszőleges HTML kóddal, miközben az eredeti HTML kód valid marad: 

	$("#select-3").divselect({
		optionTags : {
			"0"          : '<i>#</i>',
			"details"    : '<img src="../images/page.png"> <span>#</span>',
			"activate"   : '<img src="../images/lightbulb.png"> <span>#</span>',
			"inactivate" : '<img src="../images/lightbulb_off.png"> <span>#</span>',
			"modify"     : '<img src="../images/page_white_edit.png"> <span>#</span>',
			"delete"     : '<img src="../images/cross.png"> <span>#</span>'
		}
	});
HTML kód
		<select id="select-3" name="operations">
			<option value="0" selected="selected">Műveletek</option>
			<option value="details">Részletek</option>
			<option value="activate">Aktiválás</option>
			<option value="inactivate">Inaktiválás</option>
			<option value="modify">Módosítás</option>
			<option value="delete">Törlés</option>
		</select>

Használat

Illesszük be az alábbi fájlokat:

  • jQuery 1.7+
  • jQuery UI position 1.9+
  • A plugin: jquery.divselect.js
  • CSS fájl: divselect.css

Például (HTML5 kódot feltételezve): 

	<link rel="stylesheet" href="css/divselect.css" />
	<script src="js/jquery.js"></script>
	<script src="js/jquery-ui.position.js"></script>
	<script src="js/jquery.divselect.js"></script>

Készítsünk egy select elemet a HTML kódban. Itt bizonyos atribútumok befolyásolják a divselect viselkedését, ld. itt.
Fontos: az option tegek value attribútumainak egyedinek kell lennie! 

	<select id="example" name="webprog">
		<option value="html">HTML</option>
		<option value="css">CSS</option>
		<option value="js">JavaScript</option>
		<option value="php">PHP</option>
		<option value="sql">SQL</option>
	</select>

Majd a DOM betöltődése után a select elemen lefuttatjuk a plugint (itt természetesen bármilyen szelektor használható): 

	<script>
		$(document).ready(function(){
			$("#example").divselect();
		});
	</script>

További használati példák a demók között találhatóak.

Paraméterek

A konfiguráláshoz használhatóak paraméterek, melyek a létrehozáskor objektumban adhatóak át a divselect() függvénynek: 

	$("#example").divselect({
		cssPrefix : 'divselect-sort',
		showEffect: function($elem){
			$elem.slideDown();
		},
		...
	});

A paraméterek beállítása, lekérdezése és módosítása a jQuery UI widget-ekhez hasonló működési elvű. A paraméterek futási időben is módosítható az option illetve a regenerate metódusokkal (metódusokat lásd itt). A használható paraméterek az alábbiak:

Események

A divselect néhény speciális eseményéhez megadhatóak eseménykezelő függvények az alábbi módokon: 

	// inicializáláskor:
	$("#example").divselect({
		change : function(event, value, oldvalue){
			// eseménykezelő
		},
		...
	});

	// futási időben:
	$("#example").on("change.divselect", function(event, value, oldvalue){
		// eseménykezelő
	});

Minden esemény a divselect névtérhez tartozik, így csatoláskor is ezt kell megadni.
Az alapértelmezet eseménykezelő minden esetben egy üres függvény. Ha mindkét fenti módon megadtunk eseménykezelőt, az inicializáláskor megadott fog lefutni először. Az alábbi eseményekhez definiálható egyedi eseménykezelő:

Esemény Lefutás ideje Paraméterek
create A divselect elem elkészülése után fut le. Csak inicializáláskor adható meg! event
select A legördülő listában egy érték kiválasztása esetén fut le. event, value
change A kiválasztott érték módosulása esetén fut le (a fenti szűkített esete). event, value, oldvalue
open Lista lenyitásakor fut le. event
close Lista becsukásakor fut le. event

A paraméterek:
event (object): a jQuery Event objektum,
value (string): a kiválasztott option value attribútuma.
oldvalue (string): change esemény esetén az előzőleg kiválasztott option value attribútuma.
Ezenkívül az eseménykezelőben a this minden esetben az aktuális divselect elemre (legkülső div, mint DOM elem) mutat.

Metódusok

A divselect metódusai segítségével információk kérdezhetőek le, illetve manipulálható az elem futási időben. A metódusok hívása: 

	$("#example").divselect("metódusnév", paraméterek);

Azok a metódusok, melyek láncolhatóak, visszatérési értékként a divselect jQuery objektumot adják, vagyis azt, amire ráengedtük a függvényt. Erre újból ráengedhetünk egy másikat, így egyszerre többet is végrehajthatunk. Pl.: 

	$("#example").divselect("select", "name").divselect("setReadonly");

Template

A template paraméter határozza meg a select helyén felépülő HTML kód szerkezetét. Ennek a változtathatósága azért szükséges, mert a CSS kód modosítása nem mindig elegendő a megfelelő kinézet eléréséhez, néha a HTML struktúra módosítására is szükség lehet. Valamint így könnyebb egyedi funkcionalitást biztosító elemek elhelyezése a HTML kódban. A template alapesetben az alábbi szerkezetű: 

	<div class="{cssPrefix}">
		<input data-element="input" class="{cssPrefix}__input" type="hidden" name="{select.name}" value="{select.val}" />
		<div data-element="selector" class="{cssPrefix}__selector {select.readonly} {select.disabled}"></div>
		<div data-element="menu" class="{cssPrefix}__menu">
			{loop}
				{optgroup_start}
					<div data-element="group" class="{cssPrefix}__group">
						<div class="{cssPrefix}__group__title">{optgroup.label}</div>
				{/optgroup_start}
				{optgroup_end}
					</div>
				{/optgroup_end}
				{optionTags}
					<div data-element="item" class="{cssPrefix}__item {option.readonly} {option.disabled}">{option.html}</div>
				{/optionTags}
			{/loop}
		</div>
		<div data-element="background" class="{cssPrefix}__background"></div>
	</div>

A helyőrzők {...} alakúak, amik vagy lecserélődnek a megfelelő értékekre, vagy vezérlik a HTML felépítését. Csak a fenti template-ben is szereplő helyőrzők definiáltak, más helyőrzőnek nincs funkciója. Kizárólag akkor módosítsuk a template-t, ha szükséges, és akkor is a hierarchia megtartásával! Bizonyos data-element attribútummal rendelkező elemeknek speciális funkciója van:

input : ez az űrlapelem tárolja a kiválasztott option értékét.
selector : erre kattintva nyílik le az option-ök listája.
menu : a listát tároló elem.
item : a lista egy eleme, egy option-t tárol. Erre kattintva lesz az adott option kiválasztva.
background : erre kattintva csukódik be a lista.

A legkülső, {cssPrefix} class-ú a tároló elem, ennek mindenképp körbe kell vennie az egész szerkezetet.
A kattintásra működő funkciókat tilthatjuk le a disableDropdown, disableSelect és disableClose paraméterek true-ra állításával. Ezután új eseménykezelők csatolásával felüldefiniálhatjuk az alapértelmezett működést.

Billentyűvezérlés

A divselect billentyűkkel is vezérelhető. Az implementált funkciók:

Billentyű Funkció Kiváltott esemény
Tab Divselect-ek közti lépegetés (fókusz divselect-re rakása vagy leszedése) -
Kurzormozgató nyilak Fel és balra felfelé lép a listán egyesével, le és jobbra lefelé. Csukott, de fókuszt kapott lista esetén is működnek, ilyenkor a kiválasztott elemet változtatják. select, change
(csak csukott listánál)
PageUp
PageDown		 Felfelé és lefelé ugranak a listán a pagestep tulajdonságban megadott értékkel. Csukott, de fókuszt kapott lista esetén is működnek, ilyenkor a kiválasztott elemet változtatják. select, change
(csak csukott listánál)
Home
End		 A lista elejére illetve végére ugrás. Csukott, de fókuszt kapott lista esetén is működnek, ilyenkor a kiválasztott elemet változtatják. select, change
(csak csukott listánál)
Enter Nyitott lista esetén a kijelölt elem kiválasztása. select, change, close
Esc Lista becsukása, a kiválasztott elem nem változik. close
Karakterek Az a-z és 0-9 tartományba eső és a characters tulajdonságban definiált karakterek segítségével a listában gyorskeresést végezhetünk. Egy karakter leütése esetén mindig a következő, az adott karakterrel kezdődő option-re ugrik (itt az option közé írt, HTML tegek nélküli HTML kód van megvizsgálva). Ha a keydownwait tulajdonságban megadott időn belül (milliszekundum) új karaktert ütünk, a keresést pontosítjuk, vagyis ekkor a HTML kód következő karaktere is bekerül a vizsgált karakterláncba. A megadott időn túl történő leütés új keresést indít a kijelölt elem helyétől. Csukott, de fókuszt kapott lista esetén is működnek, ilyenkor a kiválasztott elemet változtatják. select, change
(csak csukott listánál)

A listákon lépegetést végző billentyűk közül a kurzormozgató és a karakterek ciklikusan működnek, vagyis a lista végére érve az elejétől folytatják.
A billentyűkkel történő vezérlést tilthatjuk le a disableKeys paraméter true-ra állításával. Ezután eseménykezelők írásával felüldefiniálhatjuk az alapértelmezett működést is. A Tab billentyű funkciója változatlan marad, mivel azt a böngésző valósítja meg, a plugin az elem "fókuszálhatóságát" biztosítja.
