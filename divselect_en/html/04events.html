Events

With the 5 event handler method, we can attach any operation to the special events of the divselect. Try the keyboard controlling also! (Tab, cursor control keys, PageUp, PageDown, Home, End, Enter, Esc)

The HTML code: 

				<select name="example1">
					<option value="1">HTML</option>
					<option value="2">CSS</option>
					<option value="3" selected="selected">JavaScript</option>
					<option value="4">PHP</option>
					<option value="5">SQL</option>
					<option value="6">Python</option>
					<option value="7">Ruby</option>
					<option value="8">ASP</option>
				</select>

The javascript code: 

				$("select").divselect({
					create : function(event){
						$("#indicator").append('The divselect has been created.');
					},
					select : function(event, value){
						var html = $(this).divselect("getselected");
						$("#indicator").append('<br />The '+value+' value has been selected ('+html+').');
					},
					change : function(event, value, oldvalue){
						var html = $(this).divselect("getselected");
						var oldhtml = $(this).divselect("getElementByValue", oldvalue).html();
						$("#indicator").append(
							'<br />The selected value has been changed: '+oldvalue+
							' ('+oldhtml+')'+
							' -&gt; '+value+
							' ('+html+').'
						);
					},
					open : function(event){
						$("#indicator").append('<br />The list has been opened.');
					},
					close : function(event){
						$("#indicator").append('<br />The list has been closed.');
					}
				});

With the other method: 

				var $DS = $("select").divselect({
					create : function(event){
						$("#indicator").append('The divselect has been created.');
					}
				});
				$DS.on("select.divselect", function(event, value){
					var html = $(this).divselect("getselected");
					$("#indicator").append('<br />The '+value+' value has been selected ('+html+').');
				});
				$DS.on("change.divselect", function(event, value, oldvalue){
					var html = $(this).divselect("getselected");
					var oldhtml = $(this).divselect("getElementByValue", oldvalue).html();
					$("#indicator").append(
						'<br />The selected value has been changed: '+oldvalue+
						' ('+oldhtml+')'+
						' -&gt; '+value+
						' ('+html+').'
					);
				});
				$DS.on("open.divselect", function(event){
					$("#indicator").append('<br />The list has been opened.');
				});
				$DS.on("close.divselect", function(event){
					$("#indicator").append('<br />The list has been closed.');
				});

There is need to set the $DS variable, because the $("select") element won't exist after the creation of the divselect, so we can't attach an event handler to that. If we use a selector that adapt to the attributes defined in the divattributes property, we don't need such method.
If we use a previous version of jQuery 1.7 we have to change the on() method call to bind() (the arguments are the same).
