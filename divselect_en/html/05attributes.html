HTML attributes

These examples show the effects of the readonly and disabled attributes.

Attributes defined in the HTML code

				<select id="example1" name="example1">
					<option value="1">HTML</option>
					<option value="2">CSS</option>
					<option value="3" readonly="readonly">JavaScript</option>
					<option value="4">PHP</option>
					<option value="5">SQL</option>
					<option value="6" disabled="disabled">Python</option>
					<option value="7" disabled="disabled">Ruby</option>
					<option value="8">ASP</option>
				</select>
				$("select").divselect({
					optiontags : {

					  "3" : '<input type="checkbox" name="lang[]" value="3" checked="checked" /> #',
					  "4" : '<input type="checkbox" name="lang[]" value="4" /> #',
					  "5" : '<input type="checkbox" name="lang[]" value="5" /> #',
					  "6" : '<input type="checkbox" name="lang[]" value="6" /> #',
					  "7" : '<input type="checkbox" name="lang[]" value="7" /> #',
					  "8" : '<input type="checkbox" name="lang[]" value="8" /> #'
					}
				});

Change the readonly and disabled states

With the buttons in the first line we can change the select state, the buttons in the second line change the third (javascript) option state.

option:

The HTML and javascript codes are the same as the above example (except there are no readonly and disabled attributes). The HTML code of the buttons and the event handlers: 

				<input id="selectdisable" type="button" value="disabled on-off" />
				<input id="selectreadonly" type="button" value="readonly on-off" />
				<input id="optiondisable" type="button" value="disabled on-off" />
				<input id="optionreadonly" type="button" value="readonly on-off" />
				$("#selectreadonly").click(function(){
					if ($("#example2").divselect("isreadonly")){
						$("#example2").divselect("unsetreadonly");
					}
					else{
						$("#example2").divselect("setreadonly");
					}
				});
				$("#optiondisable").click(function(){
					if ($("#example2").divselect("isdisabled", "3")){
						$("#example2").divselect("unsetdisabled", "3");
					}
					else{
						$("#example2").divselect("setdisabled", "3");
					}
				});
				$("#optionreadonly").click(function(){
					if ($("#example2").divselect("isreadonly", "3")){
						$("#example2").divselect("unsetreadonly", "3");
					}
					else{
						$("#example2").divselect("setreadonly", "3");
					}
				});
